podbean-roam

PodBean Roam

A chrome extension to make hosting roams with podbean easier.

This extension simply automates some of the processes necessary for hosting a roam via podbean using some javascript hacks.

Developing

Simply run yarn prod to compile the extension zip podbean-roam.zip. yarn build will build a development version.
